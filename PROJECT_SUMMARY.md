## 🚀 طرق التشغيل

### 1. التشغيل السريع (مُوصى به للاختبار)
```bash
python quick_start.py
```
- نسخة مبسطة تعمل بدون مكتبات إضافية
- واجهة أساسية لإدارة الأصول
- مثالية للاختبار السريع

### 2. التشغيل الكامل
```bash
# تثبيت المكتبات
pip install Flask Flask-SQLAlchemy Flask-Login

# تشغيل النظام الكامل
python run.py
```

### 3. تشغيل Windows
```bash
# انقر مرتين على
start.bat
```
